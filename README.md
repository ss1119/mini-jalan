## 環境構築

### 使用パッケージとそのバージョン

| パッケージ | バージョン |
| ---------- | ---------- |
| `yarn`     | 1.22.17    |
| `node`     | v16.16.0   |

### パッケージのインストール

```bash
yarn install
```

### 動作確認

```bash
npm start
```

### ディレクトリ構造

```bash
src
 ├── components         # コンポーネントを定義
 │   └── parts          # ロジックを持たないコンポーネントのみを定義
 ├── lib                # アプリケーションで用いるライブラリの再エクスポートを行う
 └── utils              # 関数や定数を管理
```
